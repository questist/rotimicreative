  • Musician Bruce C. Stevenson's Website

    About the Project

    Bruce C. Stevenson had a website already, however, his marketer wanted to create a new design for the upcoming EP release and they also needed a new developer to manage it day to day. Since Bruce is a Musician and had his own specific aesthetic's in mind such as a Monochrome palette with bright colors interspersed to catch the eye, this was the perfect opportunity to offer a custom web design solution. I showed him a prototype and he loved it, and since he knew how he wanted it to look, the minor changes we made were easy to define. I also handled the blog and will be updating it monthly with music news with the help of one of his associates.

    The Solution

    Because the site had a Blog as the main dynamic portion we opted for a custom Wordpress theme built from a Vanilla Twenty-Sixteen theme. The content was already created but the layout was drastically different from his original site. Instead of using Bootstrap for responsive functionality I opted for UIKit to change the flavour of the development process.

  • SV Family Therapy

    About the Project

    I was pleased to do this website for Mindy Dirks PhD. She had only the images for the site ready, knowing that she wanted to stick with a theme of "hands". She left the rest up to me. I created three different prototypes. One with a multi-page design and two single page variations. She decided she liked the single page designs better. After developing and coding there were a few minor changes to be made. She was very pleased with the outcome and left me a great review on Thumbtack.

    The Solution

    I used Bootstrap because it is perfect for one-page responsive designs. We only needed the grid system because the other UI elements such as the fly-in image overlays were custom coded. The colors were chosen from a three-color palette with bright accents to appeal to families and children. In the future, we may be adding dynamic content to the site including some of Mindy's writing on subjects within family Psychology.

    Link

    www.svfamilytherapy.com

  • Judy Gittelsohn - Artist Website

    About the Project

    Judy was one of the first clients I worked with on web development. She wanted a site that would allow her to manage her art herself. Because she was active in the Palo Alto art scene with exhibitions going on regularly and had collector's willing to give her social proofing we decided to emphasize that on her website. The art portfolio was built to link the exhibitions and collectors together allowing people to see which art pieces were on display at any given time.

    The Solution

    I choose Couch CMS to use for the development of the site. It offered a tight integration between the backend data and the HTML allowing us to build the design in HTML and add tags which pulled in the data. The dashboard is fully customizable and easy to use allowing Judy to manage it herself. This project was released before Wordpress had the Gallery feature built in which may have changed the decision. None the less, it was a solid choice because the site was up and running in a week. I integrated bootstrap into the website for responsive functionality.

    Link

    www.judyg.com
